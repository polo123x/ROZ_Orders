# Order Scheduler (訂單排程系統)

這是一個專為小型工廠或工作室設計的訂單排程系統，具備甘特圖視覺化介面、資源管理與雲端同步功能。

## 功能特色

*   **視覺化排程**：直觀的甘特圖介面，輕鬆查看各機台/人員的工作排程。
*   **雲端同步**：結合 Google Sheets，實現訂單與資源資料的雲端儲存與同步。
*   **資源管理**：可自訂機台或人員 (資源)，並同步至雲端。
*   **彈性工時**：支援小數點 (1.5小時) 或冒號 (1:30) 格式輸入工時。
*   **計時與提醒**：內建倒數計時器與逾期提醒功能。
*   **深色模式**：現代化的深色玻璃擬態介面 (Glassmorphism)。

## 如何使用

1.  **新增訂單**：輸入客戶名稱、內容、選擇資源、開始時間與工時。
2.  **檢視排程**：下方甘特圖會自動更新，顯示各資源的佔用情形。
3.  **管理資源**：在上方資源選單旁可新增或刪除資源。
4.  **雲端同步**：所有資料會自動儲存至 Google Sheets，重新整理網頁資料不遺失。

## 部署方式

本專案為靜態網頁，可直接部署於 GitHub Pages 或任何靜態網頁託管服務。
詳細部署教學請參閱 `deployment_guide.md`。

## 技術架構

*   Frontend: HTML5, CSS3, JavaScript (Vanilla)
*   Backend: Google Apps Script (作為 API)
*   Database: Google Sheets
